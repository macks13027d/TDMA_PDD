971 /*
972  Summary of Parameters to Control Stopping Test:
973  - Standard (default) error tolerance: |delta-residual|/|right-hand-side|<tol
974  where the norm is an energy norm wrt preconditioner, |r|=sqrt(<Cr,r>).
975  - two_norm!=0 means: the norm is the L2 norm, |r|=sqrt(<r,r>)
976  - rel_change!=0 means: if pass the other stopping criteria, also check the
977  relative change in the solution x. Pass iff this relative change is small.
978  - tol = relative error tolerance, as above
979  -a_tol = absolute convergence tolerance (default is 0.0)
980  If one desires the convergence test to check the absolute
981  convergence tolerance *only*, then set the relative convergence
982  tolerance to 0.0. (The default convergence test is <C*r,r> <=
983  max(relative_tolerance^2 * <C*b, b>, absolute_tolerance^2)
984 - cf_tol = convergence factor tolerance; if >0 used for special test
985  for slow convergence
986 - stop_crit!=0 means (TO BE PHASED OUT):
987  pure absolute error tolerance rather than a pure relative
988  error tolerance on the residual. Never applies if rel_change!=0 or atolf!=0.
989  - atolf = absolute error tolerance factor to be used _together_ with the
990  relative error tolerance, |delta-residual| / ( atolf + |right-hand-side| ) < tol
991  (To BE PHASED OUT)
992  - recompute_residual means: when the iteration seems to be converged, recompute the
993  residual from scratch (r=b-Ax) and use this new residual to repeat the convergence test.
994  This can be expensive, use this only if you have seen a problem with the regular
995  residual computation.
996  - recompute_residual_p means: recompute the residual from scratch (r=b-Ax)
997  every "recompute_residual_p" iterations. This can be expensive and degrade the
998  convergence. Use it only if you have seen a problem with the regular residual
999  computation.
1000  */
