◆ hypre_prefix_sum()

void hypre_prefix_sum ( HYPRE_Int *  in_out,
HYPRE_Int *  sum,
HYPRE_Int *  workspace 
)

Assumed to be called within an omp region. Let x_i be the input of ith thread. The output of ith thread y_i = x_0 + x_1 + ... + x_{i-1} Additionally, sum = x_0 + x_1 + ... + x_{nthreads - 1} Note that always y_0 = 0

Parameters
workspaceat least with length (nthreads+1) workspace[tid] will contain result for tid workspace[nthreads] will contain sum
