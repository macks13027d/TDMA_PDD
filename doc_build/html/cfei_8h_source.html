12 /*------------------------------------------------------------------------------
13  This is the header for the prototypes of the C interface
14  of the Finite Element Interface to Linear Solvers (FEI).
15 
16  For explanations of parameters and semantics, see the C++ FEI.h header, or
17  doxygen output created there-from.
18  With the exception of added create/destroy functions, all FEI functions
19  in this header mirror those in the C++ header except as follows:
20  - 'FEI_' is pre-pended to function names
21  - 'CFEI* cfei' is the first argument for every function
22  - arguments which are references in C++ (e.g., an output int) are
23  pointers in this C interface.
24 
25  NOTE: ALL functions return an error code which is 0 if successful,
26  non-zero if un-successful.
27 
28  Noteworthy special case: the solve function may return non-zero
29  if the solver failed to converge. This is, of course, a non-fatal
30  situation, and the caller should then check the 'status' argument for
31  possible further information (solver-specific/solver-dependent).
32 ------------------------------------------------------------------------------*/
